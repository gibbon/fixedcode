# @fixedcode/bundle-crud-api

Minimal CRUD REST API (template-only).

## Install

```bash
fixedcode registry install crud-api
```

## What it generates

A small bundle that emits a CRUD controller skeleton in TypeScript. Useful as a template starter; small surface for learning the bundle pattern.

## Spec highlights

`name`, `resource` (with attributes).
