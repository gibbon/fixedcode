# @fixedcode/bundle-spring-library

Spring Boot project skeleton (Gradle, Docker, configs).

## Install

```bash
fixedcode registry install spring-library
```

## What it generates

Generates the surrounding project: build files, Dockerfile, Spring configs, common infrastructure beans. Pair with `spring-domain` to produce a full service.

## Spec highlights

`name`, `groupId`, `artifactId`, `javaVersion`, optional `database`, `messaging` configs.
