# @fixedcode/bundle-python-agent

Python AI-agent service.

## Install

```bash
fixedcode registry install python-agent
```

## What it generates

Single-mode and orchestrator-mode AI agent on top of `python-service`: tool registry, LLM loop, HTTP endpoints.

## Spec highlights

`name`, `model.tier`, `prompt`, `tools`, optional `mode: orchestrator`.
