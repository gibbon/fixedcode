# @fixedcode/bundle-mcp-wrapper

Wraps an existing service as an MCP server.

## Install

```bash
fixedcode registry install mcp-wrapper
```

## What it generates

Generates an MCP (Model Context Protocol) server stub that proxies to existing endpoints — turn any service into a tool that LLMs can call.

## Spec highlights

`name`, `endpoints`.
