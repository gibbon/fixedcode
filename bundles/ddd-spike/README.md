# @fixedcode/bundle-ddd-spike

Original DDD reference bundle.

## Install

```bash
fixedcode registry install ddd-spike
```

## What it generates

The first bundle written for FixedCode — a stripped-down DDD layout. Kept as a reference example. For new projects, prefer `spring-domain`.
